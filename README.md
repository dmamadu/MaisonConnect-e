# Combined Angular Apple-like + i18n + Dark Mode

## Dépendances
```bash
npm i @ngx-translate/core @ngx-translate/http-loader
```

## Ce pack inclut
- Layout Apple-like (Home data JSON, pages /iphone, /mac, header, footer, directive)
- i18n (fr/en/ar) + gestion RTL
- Dark mode via Tailwind (dark class) + script anti-FOUC
- Shell minimal (plus de page Angular par défaut)
- Routes configurées

## Démarrage
- Copiez les fichiers dans votre projet Angular 18.
- Assurez-vous d'avoir Tailwind configuré.
- Lancez : `npm start` (ou SSR si configuré).
